## Installation

1. `git clone https://github.com/Red-Aura/ElianaChatBot`
2. `cd ElianaChatBot`
3. `pip3 install -r requirements.txt` to install the requirements.
4. `cp config.py`
5. Edit `config.ini` with your own values.
6. Run with `python3 eliana.py`
